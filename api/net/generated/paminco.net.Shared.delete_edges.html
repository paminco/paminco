paminco.net.Shared.delete_edges¶

Shared.delete_edges(edges, update: bool = True, **kwargs)[source]¶

Delete edge(s) from Edges object.

Parameters
edgesint or ndarray of ints

Indices to delete.

updatebool, default=True

Whether to reset mapping of node labels to node ids.

return_indices, default=False

If True, edge indices of deleted edges are returned.

Returns
ndarray, optional

If return_indices is True, edge indices of deleted nodes are returned.
